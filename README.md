# mox-sf

## Using npm

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Using bun

Bun (https://bun.sh/) is a fast Javascript runtime & package manager.

```bash
# Install dependencies
bun install

# Start development server
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
